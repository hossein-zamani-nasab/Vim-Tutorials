Vim Tutorials

i,s -> insert mode

x,X -> delete one character
o,O -> open new line
r -> replace one character
a -> append to end of the word

A -> append to end of the line
u -> undo
d -> delete
number x -> delete number of characters alongside
d + d -> delete a line

d + w -> delete a word
number dd -> delete number of lines i.e --> 3dd -> delete 3 lines

number dw -> delete number of words
J -> join

j -> scroll down
G -> go to line (G -->last line, 1G --> first line, 10G --> line10)
y -> yank/copy (yy --> one line copy, yw --> one word copy, 3y --> 3line copy and so on ...)
P,p -> paste
dd -> can also cut line and with p,P you can bring it up :)
/ whatever you want -> search (n,N -> next)
$ -> end of the line

^ -> start of the line
. -> repeat
% -> go at start/end of loop
v -> visual mode (select with arrow keys) *you can after selecting use normal mode like d -> delete
it -> inner tag (select inside tags) *in visual mode do this
v --> select --> command mode(shift :) --> sort
v --> select --> type normal(will take you to normal mode)A:(adding : to end of lines you selected)
in command mode(:) -> set number(add numbers), set nonumber(rm numbers) *note that you can click tab key to see different sets
:colorscheme -> set different colors
:Explore -> file manager

:Vexplore -> vertical file manager

:Sexplore -> split file manager
q!2 -> quit 2 times (go 2 times back)
:! -> os command like ls

:.! -> insert data here

Buffer

buffer -> in terminal you can open 2 or more files in vim with vim file1 file2 and so on

# buffer is hidden so in command mode type ls(:ls) and it shows you files you opened before * %a means your active buffer

# you can also go to next/previous buffer with :bn/bp

# also type :b1 (buffer1), :b name of your file

# deleting buffer --> :bd (if doesn't change that), :bd! (delete even change that)
